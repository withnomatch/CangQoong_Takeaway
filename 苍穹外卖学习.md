# 苍穹外卖

**nginx**

nginx反向代理：前端不直接访问后端，通过nginx代理访问

+ 提高访问速度

+ 进行负载均衡：轮询

+ 保证后端服务器安全

反向代理：proxy_pass

负载均衡：webservers+proxy_pass

**Swagger**

导入knife4j的maven坐标，加入相关配置，设置静态资源映射

**ThreadLocal**

不是一个thread，而是thread的局部变量，为每个线程提供单独一份存储空间，具有线程隔离的效果
